Program Improvements

This page summarizes how feedback from students and parents has led to real changes in the program. Each season, we review responses and take action on common themes.

2024–25 Season

Based on end-of-season feedback

Common suggestions

  • Clarify team selection criteria earlier in the process
  • Improve the progress check-in system
  • Send invitational schedules sooner

Actions taken

  • Revised team selection rubric and communicated it at the start of the season
  • Streamlined the check-in form and added reminders
  • Published invitational dates on the shared calendar as soon as they were confirmed

2023–24 Season

Based on end-of-season feedback

Common suggestions

  • More consistent communication from the leadership team
  • Clearer event assignment process for students and parent coaches

Actions taken

  • Set up a weekly email digest with updates and deadlines
  • Created an event assignment worksheet for parent coaches

Your feedback drives these improvements. Submit your thoughts through the Student or Parent Feedback forms to help shape future seasons.
